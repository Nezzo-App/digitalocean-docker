Schnellstart mit Docker

# 1. Repository klonen
git clone https://github.com/nezzo-app/nezzo.git
cd nezzo-app/docker

# 2. Umgebungsvariablen setzen
cp ../.env.example .env
# Bearbeite .env und füge deinen GROK_API_KEY ein

# 3. Starten
docker-compose up -d

# 4. Öffne http://localhost:80
