Einführung

Willkommen im Nezzo Wiki! Hier findest du umfassende Dokumentation zu allen Nezzo App Funktionen.

Nezzo App ist eine All-in-One-Plattform entwickelt von LobiGmbh (CEO) und JoviGmbh (COO).

Was ist Nezzo?

Nezzo vereint sieben mächtige Anwendungen:

  • 🧠 Nezzo AI - KI-gestützte Antworten mit Grok API
  • 💬 Nezzo Chat - Intelligente Konversationen
  • ☁️ Nezzo Cloud - Sichere Cloud-Speicherung
  • 🌐 Nezzo Host - Webhosting-Lösungen
  • 📚 Nezzo Vocab - Vokabeltrainer
  • 💻 Nezzo Code - Code-Editor & Snippets
  • 📝 Nezzo Text - Textverarbeitung

Apps Dokumentation

Nezzo AI

Nutzt die Grok API von x.ai für intelligente Textgenerierung.

POST /api/ai/generate
Body: { "prompt": "Deine Frage" }

Nezzo Chat

Führe kontextbewusste Gespräche mit der KI.

POST /api/chat/session - Neue Session erstellen
POST /api/chat/message - Nachricht senden

Nezzo Cloud

Lade Dateien sicher in die Cloud hoch.

POST /api/cloud/upload
Body: { "filename": "datei.txt", "content": "Inhalt" }

Nezzo Host

Deploye Websites mit einem Klick.

POST /api/host/deploy
Body: { "name": "Meine Seite", "content": "HTML" }

Nezzo Vocab

Erstelle Vokabelsets und lerne effizient.

POST /api/vocab/set
Body: { "name": "Englisch", "words": [...] }

Nezzo Code

Speichere und teile Code-Snippets.

POST /api/code/snippet
Body: { "title": "Hello World", "code": "...", "language": "javascript" }

Nezzo Text

Verarbeite und analysiere Texte.

POST /api/text/document
Body: { "title": "Notiz", "content": "Text" }

Docker Installation

Voraussetzungen

  • Docker & Docker Compose installiert
  • Grok API Key (von x.ai)

Installation

# Clone Repository
git clone https://github.com/nezzo-app/nezzo.git
cd nezzo-app/docker

# Umgebungsvariablen setzen
cp ../.env.example .env
# Bearbeite .env und füge deinen GROK_API_KEY ein

# Starten
docker-compose up -d

# Logs ansehen
docker-compose logs -f

Zugriff

Nach dem Start ist Nezzo App verfügbar unter: http://localhost:80

Häufige Fragen (FAQ)

Wie bekomme ich einen Grok API Key?

Registriere dich auf x.ai und erstelle einen API Key.

Ist Nezzo App kostenlos?

Ja, Nezzo App ist Open Source und kostenlos nutzbar.

Kann ich Nezzo selbst hosten?

Ja! Mit Docker Compose kannst du Nezzo einfach selbst hosten.

Wer entwickelt Nezzo?

Nezzo wird entwickelt von LobiGmbh (CEO) und JoviGmbh (COO).

Welche Sprachen werden unterstützt?

Die Oberfläche ist auf Deutsch, die KI (Grok) unterstützt multiple Sprachen.
